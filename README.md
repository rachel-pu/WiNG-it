## Getting Started

### Deploying Locally

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Next run the backend server:

```bash
cd backend
python3 app.py
```
